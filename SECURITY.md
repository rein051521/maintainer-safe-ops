# Security Policy

## Supported versions

The project is currently pre-1.0. Security fixes are applied to the latest release.

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a vulnerability

Please open a private security advisory if available, or contact the maintainer directly.

Do not publish working exploit details before maintainers have had time to review and fix the issue.

## What this tool does not guarantee

maintainer-safe-ops is a lightweight repository safety checker. It does not replace:

- full secret scanning platforms
- dependency vulnerability scanners
- manual maintainer review
- production security review
- legal or compliance review

## Safe disclosure expectations

When reporting a vulnerability, include:

- affected version or commit
- reproduction steps
- expected impact
- suggested fix if available
