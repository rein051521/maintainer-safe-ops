# Support

## Questions

Open a GitHub Discussion if Discussions are enabled. Otherwise, open an issue using the closest template.

## Bug reports

Open an issue and include:

- version
- operating system
- Python version
- command used
- expected behavior
- actual behavior
- minimal reproduction without real secrets

## Security reports

Do not open public issues for sensitive security reports. See `SECURITY.md`.

## Scope

This project provides a baseline maintainer-safety check. It does not provide legal, compliance, or production security guarantees.
